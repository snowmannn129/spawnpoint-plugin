# SpawnPoint Plugin

Official plugin/mod for [SpawnPoint](https://spawnpoint.to) — the Minecraft server listing site that keeps your listing automatically up to date.

## Downloads

See the [Releases page](https://github.com/snowmannn129/spawnpoint-plugin/releases) for all platform downloads:

- **Spigot / Paper** (1.13+) — Fully verified ✅
- **Forge / NeoForge** (1.18.2+) — Community testing 🧪
- **Fabric** (1.19+) — Community testing 🧪

## Installation

Visit your server's plugin install guide at spawnpoint.to for step-by-step instructions with your token pre-filled.
